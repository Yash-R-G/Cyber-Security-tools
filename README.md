# 🔐 Cybersecurity Tools (Python)

A Python-based cybersecurity toolkit for detecting suspicious processes, monitoring system activity, and scanning files for potential malware using basic threat detection techniques.

---

##  Features

### 🔍 Keylogger Detector
- Detects suspicious processes based on keywords
- Monitors active network connections
- Helps identify potential keylogging activity

### 🛡️ File Virus Scanner
- Scans directories for suspicious files
- Detects risky file extensions (.exe, .bat, etc.)
- Flags suspicious file names (e.g., "trojan", "virus")

---

## 🛠️ Technologies Used

- Python  
- psutil  
- OS module
